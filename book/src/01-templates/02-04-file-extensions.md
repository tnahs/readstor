# Extension

|              |                             |
| ------------ | --------------------------- |
| Name         | `extension`                 |
| Type         | string                      |
| Valid Values | any                         |
| Required     | <i class="fa fa-check"></i> |
| Default      | -                           |

Defines the template's output file extension. Note that this will be appended to
all the output filenames with the format `[filename].[extension]`.
